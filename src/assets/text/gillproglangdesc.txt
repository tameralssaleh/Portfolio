The General Interpreted Lightweight Language (GILL) is a programming language that was designed for a final project in a programming languages class in college. This language uses a tree-walking interpreter to interpret code and produce results. This project works by first passing in raw text through a lexer, which then produces tokens. At the end of the lexer stage, an array of tokens is then passed through a parser, which further processes these tokens into abstract-syntax-tree (AST) nodes. These nodes are then passed through to the interpreting stage, where they are to then be walked recursively to produce results. As of 2026, the project is still in the prototyping stage. A somewhat stable prototype of the entire system is built in Python, however, there are plans to rebuild the project in C as the final build. GILL plans to be an object-oriented language, complete with static, and enforced typing, a standard library, etc.
