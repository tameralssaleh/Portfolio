In 2022, I had ran a YouTube channel showcasing coding projects in the form of tutorials and other helpful videos. This channel grown to have just over 2,300 subscribers by 2026. This channel ultimately gained a small community of developers around it, which had then culminated into the formation of the House of Code project. House of Code aims to provide a free online space where developers of any experience levels can share their code or full projects, participate in discussion threads or forums, learn from others, etc. The first launch of the House of Code website, which is in legacy now, was discontinued in 2025 for a more performant version built in React and Express. The legacy version of this project is open-sourced and can be found on this portfolio.
